AvCoverDownloader 自动识别目录下的番号 下载大姐姐的靓照 和 本片所有出场的大姐姐姓名<br>

AvMatchCheck 自动迭代查找目录下所有的子目录的 所有视频文件 识别视频文件名称中的番号 并单独为这个视频生成一个文件夹

<br>

<br>
StartDownload 女优头像采集    请指定到Jellyfin的数据存储目录 下的 /metadata/People
